MAST PoE – Chef App
Overview

The Chef App is a web-based and mobile-friendly application built for the MAST PoE project. It allows users to explore recipes, manage ingredients, and get cooking inspiration. The project demonstrates key web development concepts such as screen navigation, modular TypeScript components, and responsive design.

Features

Home Screen – displays the main app functions and recipe categories

Recipe Display – users can view detailed recipe information

Ingredient Management – allows adding or editing ingredient lists

Responsive Design – optimized for desktop and mobile devices

TypeScript (TSX) – provides type safety and modular code organization

Navigation – smooth transitions between screens and sections

Error Handling – handles missing data or invalid inputs gracefully

Technologies Used

Frontend: React + TypeScript (TSX)

Styling: CSS / Tailwind (depending on your setup)

Runtime: Node.js

Package Manager: npm or yarn

Installation & Setup

Clone the repository

git clone https://github.com/your-username/mast-chef-app.git


Navigate to the project folder

cd mast-chef-app


Install dependencies

npm install


Run the app

npx expo start --web


Open in browser
Visit http://localhost:3000 (or the provided local URL)

This project demonstrates:

Application of React and TypeScript fundamentals

Implementation of navigation between app sections

Proper use of components and modular design

Effective error handling and user interface design

Understanding of responsive layout techniques
